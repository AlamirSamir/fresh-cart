## ✨ Features

- User Authentication (Login & Register)
- Browse Products
- Browse Categories
- Browse Brands
- Product Details
- Shopping Cart Management
- Wishlist Management
- Responsive Design
- Protected Routes (Route Guards)
- Reactive Forms & Validation
- RESTful API Integration

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/AlamirSamir/Fresh_Cart.git
```

Go to the project folder

```bash
cd Fresh_Cart
```

Install dependencies

```bash
npm install
```

Run the project

```bash
ng serve
```

Open

```
http://localhost:4200
```
